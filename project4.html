Cosmic Social

This is a social media application that allows the user to create a profile, share text and/or image posts, follow other users, like other users post,view all posts ,and search for other users. The front-end is made using React JS, Redux, Material-UI, and Bootstrap 4. The back-end is made using Express in Node JS, AWS S3, and MongoDB.
